## Step 1 Guide

First, run backend by entering into backend folder with

```bash
cd backend

npm install

yarn install

pnpm install

bun install
```

Then run the backend development server with:

```bash
npm run start:dev
# or
yarn start:dev
# or
pnpm start:dev
# or
bun start:dev
```

## Step 2 Guide

Run Frontend by entering into frontend folder with

```bash
cd frontend

npm install

yarn install

pnpm install

bun install

```

Then run the development server with:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
